## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/THAJITHA/kodbud.git
```

2. Navigate to the project:
```bash
cd kodbud
```

3. Open any task folder in your preferred Java IDE.

4. Compile the Java file:
```bash
javac Main.java
```

5. Run the program:
```bash
java Main
```
